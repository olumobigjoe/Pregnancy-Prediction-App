About the project

Clinical AI for
Reproductive Medicine

Assisted Reproductive Technology (ART) — particularly In Vitro Fertilisation (IVF) — is one of the most emotionally and clinically complex areas of modern medicine. Predicting which patients will achieve a successful pregnancy helps clinicians counsel patients, allocate resources, and design better treatment protocols.

This project applies a Random Forest Classifier to 604 real IVF patient records collected over 11 years (2010–2021), using six clinical and embryological features to predict the binary outcome: pregnancy achieved (1) or not (0).

The trained model is deployed as a Streamlit web application, providing an interactive interface where clinicians or researchers can input patient parameters and receive an instant probability-based prediction with confidence scores.

D5 blastocyst (91.9%) vs D3 cleavage stage (8.1%). Encoded: D3→0, D5→1. Blastocysts generally show higher implantation rates clinically.

Morphological grade: Grade 1 (best) to Grade 4. 88.2% of embryos graded 2. Contributes 5.4% importance. Encoded ordinally (0–4).

Streamlit deployment

Live web app with
probability output

The trained model is deployed as a Streamlit application with a clean two-column layout. Clinicians enter patient parameters using dropdown selects (for categorical features) and number inputs. On prediction, the app returns both a classification result and the raw probability score for both classes.

Categorical features use st.selectbox with human-readable labels
Input fed as a named pd.DataFrame — resolves sklearn feature name validation

# Input as named DataFrame — exact column order
input_data = pd.DataFrame(
    [[Age, BMI, No_Embryos,
      Type_Embryo[1],
      Embryo_Quality[1],
      Sperm_Quality[1]]],
    columns=[
        'Age', 'BMI',
        'No_Embryos_Transferred',
        'Type_Embryo',
        'Embryo_Quality',
        'Sperm_Quality'
    ]
)

prediction  = model.predict(input_data)[0]
probability = model.predict_proba(input_data)[0]

if prediction == 1:
    st.success("Pregnancy Likely")
    st.metric("Probability",
             f"{probability[1]*100:.1f}%")
else:
    st.error("Pregnancy Unlikely")

Future work

What comes next

High
Address class imbalance with SMOTE or class_weight
The 76/24 class split suppresses recall on the pregnancy class to 20.7%. SMOTE oversampling or setting class_weight='balanced' would directly improve sensitivity.
High
Hyperparameter tuning via GridSearchCV
Optimise n_estimators, max_depth, min_samples_leaf, and max_features to reduce overfitting and improve AUC.
Medium
Incorporate ovarian reserve markers and stimulation protocol
AMH levels, antral follicle count, and stimulation protocol are clinically proven IVF predictors that exist in the dataset but were not included in this model.
Medium
Compare with XGBoost, SVM, and Logistic Regression
Benchmark the Random Forest against gradient boosting and linear classifiers to identify the best-performing algorithm on this dataset.
Medium
Add SHAP explainability to the Streamlit app
Display a patient-level SHAP waterfall chart showing which features drove each individual prediction — critical for clinical trust and interpretability.
Low
Time-series trend analysis (2010–2021)
Analyse whether clinic success rates improved over the 11-year span and whether feature importance shifted over time — valuable for clinical audit.
